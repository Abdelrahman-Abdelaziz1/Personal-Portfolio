# 🚀 Professional Portfolio | Data Science & Full-Stack Development

A high-performance personal brand platform developed using  Next.js 15 ,  Tailwind CSS , and  Framer Motion  to showcase dual-track expertise in Data Science and Web Development.

## 🌟 Overview

This project serves as a centralized hub for my professional identity, highlighting my journey as an MSc Data Science student at TU Darmstadt and my experience as a Full-Stack Developer. The application is optimized for performance and features a modern, premium UI/UX.

## 🛠️ Tech Stack

### Frontend & Design
-  Framework:  Next.js 15 (App Router)
-  Styling:  Tailwind CSS 
-  Animations:  Framer Motion & Aceternity UI 
-  Components:  Shadcn UI & Chakra UI 
-  State Management:  Zustand & React Query 

### Deployment & Tools
-  Platform:  Vercel 
-  Version Control:  Git
-  Language:  TypeScript

## ✨ Key Features

-  Dual-Track Showcase:  Integrated sections highlighting both Data Science analytical projects and Web Development technical builds.
-  Interactive CV Viewer:  A premium "Glassmorphism" style viewer for professional credentials.
-  Dynamic Project Galleries:  Filterable and interactive displays for portfolio pieces.
-  Responsive Design:  Fully optimized for mobile, tablet, and desktop views with seamless dark/light mode transitions.
-  Performance Optimized:  Built with Next.js best practices to ensure fast loading times and smooth navigation.
